# Milestone 4 — First Video Bring-Up: Single VDP, Graphic 4

Objective:
- Render the first documented useful frame path: one V9938 in Graphic 4 with
  palette output and vertical scroll.

Required docs:
- `docs/spec/00-overview.md`
- `docs/spec/02-video.md`
- `docs/emulator/04-video.md`
- `docs/emulator/07-implementation-plan.md`

Allowed scope:
- VDP port interface, VRAM/palette storage, Graphic 4 renderer, `R#23`
  vertical scroll, SDL/OpenGL upload, and headless frame dumps.

Explicit non-goals:
- Horizontal scroll via `R#26`/`R#27`
- Dual-VDP compositing
- Command engine implementation

Allowed paths:
- `src/core/video/`
- `src/frontend/`
- `shaders/`
- `tests/`

Required tests:
- VDP port writes affect VRAM output.
- Palette decoding matches spec.
- Graphic 4 addressing across scanlines.
- `R#23` vertical scroll wrapping.

Exit criteria:
- A known frame matches in headless and SDL/OpenGL paths.

Forbidden extras:
- Unverified scroll behavior or mixed-mode support.

Verification commands:
```bash
cmake -B build -DCMAKE_TOOLCHAIN_FILE=third_party/vcpkg/scripts/buildsystems/vcpkg.cmake
cmake --build build
ctest --test-dir build --output-on-failure
```
