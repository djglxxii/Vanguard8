# Milestone 11 — Coverage Expansion, Audit Closure, and 1.0 Readiness

Objective:
- Finish the documented hardware surface, close known assumptions, and prepare
  a releasable baseline.

Required docs:
- `docs/spec/00-overview.md`
- `docs/spec/01-cpu.md`
- `docs/spec/02-video.md`
- `docs/spec/03-audio.md`
- `docs/spec/04-io.md`
- `docs/emulator/04-video.md`
- `docs/emulator/06-debugger.md`
- `docs/emulator/07-implementation-plan.md`

Allowed scope:
- Additional documented VDP modes, mixed-mode validation where specified,
  remaining debugger panels, performance/correctness fixes, compatibility
  audit notes, and documentation closure.

Explicit non-goals:
- Implementing any explicitly deferred spec gaps without a spec update.

Allowed paths:
- `src/`
- `tests/`
- `docs/`

Required tests:
- Mode-specific rendering coverage for each newly implemented VDP mode.
- Cross-subsystem integration tests.
- Long-running determinism and save/replay stress coverage.

Exit criteria:
- The documented hardware surface is implemented except for explicit
  spec deferrals.

Forbidden extras:
- Invented behavior for unresolved scroll, 4-player, or cartridge SRAM gaps.

Verification commands:
```bash
cmake -B build -DCMAKE_TOOLCHAIN_FILE=third_party/vcpkg/scripts/buildsystems/vcpkg.cmake
cmake --build build
ctest --test-dir build --output-on-failure
```
