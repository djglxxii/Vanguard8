# Showcase Assets

Place source assets for the showcase ROM here.

Version 1 assumes these assets are generated by coding agents or assembled from
simple procedural/authored primitives. They should be coherent and reviewable,
not elaborate.

Recommended layout:
- `art/` source stills, logos, sprite sheets, and tile references
- `audio/` music references, generated WAVs, and chip-programming source data
- `provenance/` prompt/spec notes, palette assumptions, and conversion intent
  for every generated source asset

Do not check converted runtime blobs directly into this folder unless the
conversion step is intentionally part of source control.
