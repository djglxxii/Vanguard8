# Milestone 0 — Project Skeleton and Verification Harness

Objective:
- Create the build, target, source-tree, and test harness scaffold described in
  `docs/emulator/01-build.md`.

Required docs:
- `docs/emulator/01-build.md`
- `docs/emulator/07-implementation-plan.md`

Allowed scope:
- Add the top-level build files and directory layout.
- Add `vanguard8_core`, `vanguard8_frontend`, and `vanguard8_headless`
  targets.
- Add Catch2 wiring, logging, config loading, and a no-op emulator shell.

Explicit non-goals:
- Bus, CPU, video, audio, cartridge, or controller emulation.
- Debugger implementation beyond empty target wiring.

Allowed paths:
- `CMakeLists.txt`
- `vcpkg.json`
- `vcpkg-configuration.json`
- `src/`
- `tests/`
- `third_party/`
- `shaders/`

Required tests:
- Configure/build succeeds from a clean checkout.
- `ctest` runs successfully.
- Headless binary starts, prints version/build info, and exits cleanly.

Exit criteria:
- The repo matches the documented scaffold and is testable.

Forbidden extras:
- Emulator subsystem behavior beyond a no-op shell.
- Milestone 1 memory or port routing work.

Verification commands:
```bash
cmake -B build -DCMAKE_TOOLCHAIN_FILE=third_party/vcpkg/scripts/buildsystems/vcpkg.cmake
cmake --build build
ctest --test-dir build --output-on-failure
```
