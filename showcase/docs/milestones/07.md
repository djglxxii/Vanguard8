# Showcase Milestone 7 — Scrolling Starfield and Graphic 6 HUD Overlay

Objective:
- Add one final attract-style scene that demonstrates mixed-mode dual-VDP
  operation with a vertically scrolling `Graphic 4` background and a fixed
  `Graphic 6` high-resolution HUD overlay.

Required docs:
- `docs/spec/02-video.md`
- `docs/spec/04-io.md`
- `showcase/docs/showcase-rom-plan.md`
- `showcase/docs/showcase-implementation-plan.md`
- `docs/emulator/milestones/16.md`

Allowed scope:
- One additional banked showcase scene
- A compact bitmap glyph/icon set for HUD text and indicators
- Updated checkpoint manifests and review notes for the new scene

Explicit non-goals:
- `Text 1`/`Text 2` showcase content
- `Graphic 5` or `Graphic 7` content
- Horizontal scroll or parallax via `R#26`/`R#27`
- Emulator-core work

Allowed paths:
- `showcase/`

Required verification:
- The ROM still builds through the existing showcase wrapper
- A documented headless command reaches the new late-loop scene and emits a
  stable frame hash
- Review notes explain the expected separation between the scrolling
  background layer and the fixed HUD layer

Exit criteria:
- A reviewer can confirm from the documented command and manifest that the
  starfield scrolls vertically while the `Graphic 6` HUD remains fixed,
  transparent where intended, and legible.

Forbidden extras:
- Turning the milestone into a new engine/toolkit pass
- Adding new audio systems, replay infrastructure, or unrelated showcase scenes

Verification commands:
```bash
python3 showcase/tools/package/build_showcase.py
python3 showcase/tools/package/run_showcase_headless.py --frames 1680 --trace build/showcase/m7.trace --hash-frame 1679 --symbols build/showcase/showcase.sym
```
