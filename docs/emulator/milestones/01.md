# Milestone 1 — Memory Map, Cartridge, SRAM, and Bus

Objective:
- Implement physical memory routing, cartridge banking, SRAM, and stubbed I/O
  endpoints before CPU execution.

Required docs:
- `docs/spec/00-overview.md`
- `docs/emulator/03-cpu-and-bus.md`
- `docs/emulator/07-implementation-plan.md`

Allowed scope:
- Add the `Bus`, cartridge, SRAM, open-bus behavior, and documented port stubs.

Explicit non-goals:
- CPU execution, MMU translation, interrupt dispatch, scheduler, video, or
  audio logic.

Allowed paths:
- `src/core/`
- `tests/`

Required tests:
- Fixed ROM, banked ROM, and SRAM physical mappings.
- `0xFF` behavior for unmapped memory and I/O.
- 960 KB cartridge size validation.

Exit criteria:
- Bus routing matches `docs/spec/00-overview.md`.

Forbidden extras:
- Any CPU core integration or boot-sequence logic.

Verification commands:
```bash
cmake -B build -DCMAKE_TOOLCHAIN_FILE=third_party/vcpkg/scripts/buildsystems/vcpkg.cmake
cmake --build build
ctest --test-dir build --output-on-failure
```
